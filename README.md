# Twitter Clone

Projeto feito utilizando o miniframework MVC construído no curso da Udemy, este projeto simula algumas das funções do Twitter como seguir usuários, fazer
tweets dentre outros.

### 🔧 Para iniciar o miniframework
* 1º - Ative o XAMPP
